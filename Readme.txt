Данный репозиторий содержит материалы курса по ИИ.

В нём 4 раздела:

1. DS содержит материалы, посвящённые анализу данных;
2. ML содержит материалы, посвящённые классическим алгоритмам ML;
3. NN содержит материалы, посвящённый нейронным сетям;
4. Final_Attestation содержит итоговый проект.

Большинство тем содержат файл с учебными материалами и файл с лабораторной работой.
